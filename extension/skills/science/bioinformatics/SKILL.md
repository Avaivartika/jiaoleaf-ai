# Bioinformatics Skill

Use this skill for computational biology and omics manuscripts.

## Review Focus

- Validate that datasets, cohorts, preprocessing, normalization, and quality control are described.
- Check whether batch effects, multiple testing, leakage, and confounders are handled.
- For single-cell analysis, look for clustering method, marker validation, annotation rationale, and integration strategy.
- For sequence models, check train/test split leakage, homology controls, and evaluation metrics.

## Output

- Identify missing methods details.
- Flag biological claims that exceed the computational evidence.
- Suggest concrete manuscript edits and additional checks.
