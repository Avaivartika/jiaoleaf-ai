# Rebuttal Response Skill

Use this skill for reviewer response and revision planning.

## Workflow

1. Classify each reviewer point as factual correction, missing experiment, writing clarity, novelty concern, or disagreement.
2. Draft a direct answer that acknowledges the concern.
3. State the concrete manuscript change or experiment.
4. Keep tone factual and non-defensive.
5. Track unresolved issues separately.

## Output

- Provide a point-by-point response.
- Include suggested manuscript edits when useful.
- Avoid promising experiments or results that are not already available.
