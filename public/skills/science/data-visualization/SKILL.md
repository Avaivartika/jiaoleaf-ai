# Data Visualization Skill

Use this skill to improve figures, tables, and captions in a manuscript.

## Guidance

- Align each figure with one message.
- Prefer direct labels over legends when space allows.
- Check whether axes, units, uncertainty, sample size, and color meaning are explicit.
- Use colorblind-safe encodings and avoid relying on color alone.
- Make captions self-contained: what is shown, how it was produced, and what conclusion to draw.

## Output

- Provide figure-by-figure issues.
- Suggest caption rewrites.
- If editing LaTeX, preserve existing figure labels and references.
